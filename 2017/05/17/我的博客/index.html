5. 源代码引用

源代码引用,只需要段前和段后有空行,并且源码的每一行前面都有制表符(\t),或者4个空格的缩进就会被解析为源码原样引用显示. 类似于 html 的 code, pre 标签.

另一种行内源码的表示方式是使用键盘上方数字键那一行最左边,数字1左边的 ` 符号括起来即可,注意不是单引号.
